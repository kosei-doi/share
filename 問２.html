微分積分 問2 解答

【問題】

制約条件 $φ(x,y) = x^3 + y^3 + 2x + 2y - 6 = 0$ の下で、関数 $f(x,y)=2xy$ の最大値、最小値を調べよ。

【方針】
必要な知識・テクニック
  • ラグランジュの未定乗数法：制約条件の下で関数の極値を求める際に使用する方法です。目的関数 $f(x,y)$ と制約条件 $g(x,y)=0$ に対して、ラグランジュ関数 $L(x,y,\lambda) = f(x,y) - \lambda g(x,y)$ を定義し、その偏導関数がすべて $0$ となる点を求めます。ここで、$\lambda$ はラグランジュ乗数（未定乗数）と呼ばれます。
  • 制約条件：問題では、$(x,y)$ が $φ(x,y) = x^3 + y^3 + 2x + 2y - 6 = 0$ を満たす必要があります。この制約条件を満たす点の集合（曲線）上で、目的関数 $f(x,y)$ の最大値・最小値を求めます。
  • 停留点の必要条件：制約条件の下での極値候補点は、ラグランジュ関数の偏導関数がすべて $0$ となる点です。すなわち、$\frac{\partial L}{\partial x} = 0$、$\frac{\partial L}{\partial y} = 0$、$\frac{\partial L}{\partial \lambda} = 0$ を満たす点が候補となります。
  • 最大値・最小値の判定：停留点における関数値を比較し、最大値・最小値を決定します。また、制約曲線が有界でない場合は、無限遠での関数の挙動も調べる必要があります。

【解答】

この問題では、ラグランジュの未定乗数法を用いて、制約条件 $φ(x,y) = x^3 + y^3 + 2x + 2y - 6 = 0$ の下で関数 $f(x,y) = 2xy$ の最大値・最小値を求めます。まず、ラグランジュ関数を定義し、その偏導関数を計算して停留点を求めます。次に、各停留点における関数値を比較し、制約曲線の有界性を確認して最大値・最小値を決定します。

（1）ラグランジュ関数の定義

制約条件 $φ(x,y) = x^3 + y^3 + 2x + 2y - 6 = 0$ の下で、目的関数 $f(x,y) = 2xy$ の極値を求めるために、ラグランジュ関数を定義します。

$$L(x,y,\lambda) = f(x,y) - \lambda φ(x,y) = 2xy - \lambda(x^3 + y^3 + 2x + 2y - 6)$$

ここで、$\lambda$ はラグランジュ乗数（未定乗数）と呼ばれる実数です。この関数の停留点を求めることで、制約条件の下での目的関数の極値候補点が得られます。

ラグランジュの未定乗数法は、制約条件付き最適化問題を解くための強力な方法です。制約条件を満たしながら目的関数を最大化・最小化する点では、目的関数の勾配と制約条件の勾配が平行になるという幾何学的な直観に基づいています。
（2）ラグランジュ関数の偏導関数の計算

ラグランジュ関数 $L(x,y,\lambda) = 2xy - \lambda(x^3 + y^3 + 2x + 2y - 6)$ の各変数に関する偏導関数を計算します。

まず、$L$ を $x$ で偏微分します。$2xy$ の $x$ に関する偏導関数は $2y$（$y$ は定数として扱う）、$-\lambda(x^3 + y^3 + 2x + 2y - 6)$ の $x$ に関する偏導関数は $-\lambda(3x^2 + 2)$ となります。

$$\frac{\partial L}{\partial x} = 2y - \lambda(3x^2 + 2)$$

次に、$L$ を $y$ で偏微分します。$2xy$ の $y$ に関する偏導関数は $2x$（$x$ は定数として扱う）、$-\lambda(x^3 + y^3 + 2x + 2y - 6)$ の $y$ に関する偏導関数は $-\lambda(3y^2 + 2)$ となります。

$$\frac{\partial L}{\partial y} = 2x - \lambda(3y^2 + 2)$$

最後に、$L$ を $\lambda$ で偏微分します。$2xy$ は $\lambda$ を含まないため導関数は $0$、$-\lambda(x^3 + y^3 + 2x + 2y - 6)$ の $\lambda$ に関する偏導関数は $-(x^3 + y^3 + 2x + 2y - 6)$ となります。これは制約条件 $φ(x,y) = 0$ を表しています。

$$\frac{\partial L}{\partial \lambda} = -(x^3 + y^3 + 2x + 2y - 6) = -φ(x,y)$$
（3）連立方程式を解いて停留点を求める

停留点を求めるため、すべての偏導関数が $0$ となる連立方程式を解きます。

$$\begin{cases} \frac{\partial L}{\partial x} = 2y - \lambda(3x^2 + 2) = 0 \\ \frac{\partial L}{\partial y} = 2x - \lambda(3y^2 + 2) = 0 \\ \frac{\partial L}{\partial \lambda} = -(x^3 + y^3 + 2x + 2y - 6) = 0 \end{cases}$$

第1式と第2式から、$\lambda$ について解きます。第1式より、$\lambda \neq 0$ の場合（$3x^2 + 2 \neq 0$ は常に成り立つ）、

$$\lambda = \frac{2y}{3x^2 + 2}$$

第2式より、

$$\lambda = \frac{2x}{3y^2 + 2}$$

これらが等しいことから、

$$\frac{2y}{3x^2 + 2} = \frac{2x}{3y^2 + 2}$$

両辺に $(3x^2 + 2)(3y^2 + 2)$ をかけると、

$$2y(3y^2 + 2) = 2x(3x^2 + 2)$$ $$6y^3 + 4y = 6x^3 + 4x$$ $$6(y^3 - x^3) = 4(x - y)$$ $$6(y - x)(y^2 + xy + x^2) = 4(x - y)$$ $$6(y - x)(y^2 + xy + x^2) = -4(y - x)$$

$y - x = 0$ または $6(y^2 + xy + x^2) = -4$ となります。

第2の可能性 $6(y^2 + xy + x^2) = -4$ について、左辺は $y^2 + xy + x^2 = (y + \frac{x}{2})^2 + \frac{3x^2}{4} \geq 0$ であるため、これは成り立ちません。

したがって、$y - x = 0$ すなわち $y = x$ が成り立ちます。

これを第3式（制約条件）に代入すると、

$$x^3 + x^3 + 2x + 2x - 6 = 0$$ $$2x^3 + 4x - 6 = 0$$ $$x^3 + 2x - 3 = 0$$

この三次方程式を解きます。$x = 1$ を代入すると、$1 + 2 - 3 = 0$ となるため、$x = 1$ は解です。

因数定理により、$x^3 + 2x - 3 = (x - 1)(x^2 + x + 3)$ と因数分解できます。$x^2 + x + 3$ の判別式は $1 - 12 = -11 < 0$ であるため、実数解は $x = 1$ のみです。

したがって、停留点は $(x, y) = (1, 1)$ のみです。

このとき、$\lambda$ の値は、第1式より、

$$\lambda = \frac{2y}{3x^2 + 2} = \frac{2 \cdot 1}{3 \cdot 1^2 + 2} = \frac{2}{5}$$

となります。

（4）停留点における関数値の計算

停留点 $(1, 1)$ における目的関数 $f(x,y) = 2xy$ の値を計算します。

$$f(1, 1) = 2 \cdot 1 \cdot 1 = 2$$

また、制約条件が満たされることを確認します。

$$φ(1, 1) = 1^3 + 1^3 + 2 \cdot 1 + 2 \cdot 1 - 6 = 1 + 1 + 2 + 2 - 6 = 0$$

したがって、停留点 $(1, 1)$ において、$f(1, 1) = 2$ となります。

（5）制約曲線の有界性と無限遠での挙動

制約条件 $φ(x,y) = x^3 + y^3 + 2x + 2y - 6 = 0$ について、$x \to +\infty$ や $x \to -\infty$ のときの挙動を調べます。

制約条件 $x^3 + y^3 + 2x + 2y - 6 = 0$ より、$y^3 = -x^3 - 2x - 2y + 6$ となります。$x$ が非常に大きいとき、$y^3 \approx -x^3$ となり、$y \approx -x$ となります（実数の3乗根を考えると）。

同様に、$x \to -\infty$ のときも、$y^3 \approx -x^3$ となり、$y \approx -x$ となります。

したがって、$x \to +\infty$ のとき $y \to -\infty$、$x \to -\infty$ のとき $y \to +\infty$ となります。制約曲線は有界ではありません。

制約条件を満たす点において、目的関数 $f(x,y) = 2xy$ について、$x \to +\infty$、$y \to -\infty$ のとき、$f(x,y) = 2xy \to -\infty$ となります。同様に、$x \to -\infty$、$y \to +\infty$ のときも、$f(x,y) = 2xy \to -\infty$ となります。

したがって、制約条件の下では関数 $f(x,y)$ には最大値も最小値も存在しません。停留点 $(1, 1)$ における関数値 $f(1, 1) = 2$ は、局所的な極値ではありますが、最大値でも最小値でもありません。

制約曲線が有界でない場合、停留点が存在しても、それよりも大きい（または小さい）関数値を持つ点が無限遠に存在する可能性があります。このような場合は、最大値または最小値が存在しないことがあります。
【検証】

まず、求めた停留点が制約条件を満たすことを確認します。点 $(1, 1)$ について、制約条件 $φ(x,y) = x^3 + y^3 + 2x + 2y - 6$ に代入すると、$φ(1, 1) = 1^3 + 1^3 + 2 \cdot 1 + 2 \cdot 1 - 6 = 1 + 1 + 2 + 2 - 6 = 0$ となり、確かに制約条件を満たすことを確認できます。

次に、ラグランジュ関数の偏導関数が $0$ になることを確認します。点 $(1, 1)$ について、$\lambda = \frac{2}{5}$ とすると、$\frac{\partial L}{\partial x} = 2 \cdot 1 - \frac{2}{5}(3 \cdot 1^2 + 2) = 2 - \frac{2}{5} \cdot 5 = 2 - 2 = 0$、$\frac{\partial L}{\partial y} = 2 \cdot 1 - \frac{2}{5}(3 \cdot 1^2 + 2) = 2 - \frac{2}{5} \cdot 5 = 2 - 2 = 0$ となり、偏導関数が $0$ になることを確認できます。

最後に、無限遠での挙動を確認します。例えば、$x = 100$ のとき、制約条件より $y^3 \approx -x^3 = -10^6$ となるため、$y \approx -100$ となります。このとき、$f(100, -100) = 2 \cdot 100 \cdot (-100) = -20000$ となり、$f(1, 1) = 2$ よりも小さい値をとります。同様に、$x \to -\infty$ のときも $f(x,y) \to -\infty$ となるため、最小値は存在しないことを確認できます。

以上より求めた結果の正しさを確認しました。

【答】

制約条件 $φ(x,y) = x^3 + y^3 + 2x + 2y - 6 = 0$ の下で、関数 $f(x,y) = 2xy$ の極値は以下の通りです。

  • 停留点：$(1, 1)$ で $f(1, 1) = 2$
  • 最大値：存在しない（$x \to \pm\infty$ のとき $f(x,y) \to -\infty$ となるため）
  • 最小値：存在しない（$x \to \pm\infty$ のとき $f(x,y) \to -\infty$ となるため）

停留点 $(1, 1)$ における関数値 $f(1, 1) = 2$ は局所的な極値ですが、制約曲線が有界でないため、最大値でも最小値でもありません。
