【問題】

関数 $f(x,y)=x^{4}+y^{4}-x^{2}-2xy-y^{2}+3$ の極値を調べよ。

【方針】
必要な知識・テクニック
  • 停留点：$f_{x}=0$ かつ $f_{y}=0$ を満たす点を求めます。極値は停留点でのみ起こり得ます。
  • 2階偏導関数と判別式：$f_{xx}, f_{yy}, f_{xy}$ を計算し、$D=(f_{xy})^{2}-f_{xx}f_{yy}$ を用いて極値判定を行います。
  • 判別式の判定：$D<0$ のとき $f_{xx}$ の符号で極大・極小を判定（$f_{xx}>0$ で極小、$f_{xx}<0$ で極大）。$D>0$ のとき鞍点（極値でない）。$D=0$ のとき判別式では決まらず、定義に戻って調べます。
  • 代入と符号チェック：$D=0$ の場合、近傍の関数値を具体的な経路で比べ、極値か否かを判断します。
  • 多項式の因数分解：共通因数をくくり、$x^{3}-y^{3}=(x-y)(x^{2}+xy+y^{2})$ を利用して停留点を整理します。
開始
関数と偏導関数を確認する
ステップ1
$f_{x}, f_{y}$ を計算し、停留点を求める
ステップ2
2階偏導関数 $f_{xx}, f_{yy}, f_{xy}$ を計算する
ステップ3
判別式 $D=(f_{xy})^{2}-f_{xx}f_{yy}$ を求める
判定
$D$ と $f_{xx}$ の符号で極値を判定（$D=0$ は別途検討）
終了
各停留点の極大・極小・非極値をまとめる

【解答】

この問題では、まず1階偏導関数を計算して停留点を求め、次に2階偏導関数と判別式 $D$ を使って極値かどうかを判定します。$D=0$ の場合は、定義に戻って近傍の値を比較し、極値か否かを判断します。

（1）偏導関数と停留点の計算

関数 $f(x,y)=x^{4}+y^{4}-x^{2}-2xy-y^{2}+3$ の1階偏導関数を求めます。

$x$ について偏微分します。$x^{4}$ から $4x^{3}$、$-x^{2}$ から $-2x$、$-2xy$ から $-2y$ が出ます。$y^{4}$ と $-y^{2}$ と $+3$ は $x$ を含まないため 0 です。

$$f_{x}(x,y)=4x^{3}-2x-2y$$

$y$ について偏微分します。$y^{4}$ から $4y^{3}$、$-y^{2}$ から $-2y$、$-2xy$ から $-2x$ が出ます。$x^{4}$ と $-x^{2}$ と $+3$ は $y$ を含まないため 0 です。

$$f_{y}(x,y)=4y^{3}-2x-2y$$

停留点は $f_{x}=0$ かつ $f_{y}=0$ を満たす点です。まず $f_{x}=f_{y}$ を比較して差をとると、

$$4x^{3}-2x-2y = 4y^{3}-2x-2y \implies 4(x^{3}-y^{3})=0 \implies x^{3}-y^{3}=0 \implies x=y$$

となり、停留点は対角線 $x=y$ 上にあります。

$x=y$ を $f_{x}=0$ に代入します。$4x^{3}-2x-2x=0$ より

$$4x^{3}-4x=0 \implies 4x(x^{2}-1)=0 \implies x=0,\ 1,\ -1$$

となります。$x=y$ なので停留点は $(0,0)$、$(1,1)$、$(-1,-1)$ です。

補足： $x^{3}-y^{3}=(x-y)(x^{2}+xy+y^{2})$ から $x=y$ を得てもよいです。$x=y$ で整理すると $x^{2}-1=0$ となり、$x=\pm1$ と $0$ が得られます。
（2）2階偏導関数と判別式

2階偏導関数を計算します。

$f_{xx}$：$f_{x}=4x^{3}-2x-2y$ を $x$ で偏微分し、$12x^{2}-2$。

$$f_{xx}(x,y)=12x^{2}-2$$

$f_{yy}$：$f_{y}=4y^{3}-2x-2y$ を $y$ で偏微分し、$12y^{2}-2$。

$$f_{yy}(x,y)=12y^{2}-2$$

$f_{xy}$：$f_{x}$ を $y$ で偏微分し、$-2$。

$$f_{xy}(x,y)=-2$$

判別式 $D$ は

$$D=(f_{xy})^{2}-f_{xx}f_{yy}=(-2)^{2}-(12x^{2}-2)(12y^{2}-2)$$

展開すると

$$D=4-(144x^{2}y^{2}-24x^{2}-24y^{2}+4)=-144x^{2}y^{2}+24x^{2}+24y^{2}=24(-6x^{2}y^{2}+x^{2}+y^{2})$$
（3）各停留点の判定と値

各停留点で $D$ と $f_{xx}$ を評価し、極値かどうかを判定します。

$(0,0)$：$D(0,0)=0$。判別式では判定できないので、近傍の値を経路で比較します。$f(0,0)=3$ です。

まず、$f(x,y)$ を変形します。$-x^{2}-2xy-y^{2} = -(x^{2}+2xy+y^{2}) = -(x+y)^{2}$ であるため、

$$f(x,y) = x^{4}+y^{4}-(x+y)^{2}+3$$

となります。

次に、$y=-x$ という経路で $(0,0)$ に近づける場合を考えます。$y=-x$ を代入すると、$(x+y)^{2} = (x+(-x))^{2} = 0^{2} = 0$ であるため、

$$f(x,-x) = x^{4}+(-x)^{4}-0+3 = x^{4}+x^{4}+3 = 2x^{4}+3$$

となります。$x \ne 0$ のとき $x^{4} > 0$ であるため、$f(x,-x) = 2x^{4}+3 > 3 = f(0,0)$ です。

一方、$y=x$ という経路で $(0,0)$ に近づける場合を考えます。$y=x$ を代入すると、$(x+y)^{2} = (2x)^{2} = 4x^{2}$ であるため、

$$f(x,x) = x^{4}+x^{4}-(2x)^{2}+3 = 2x^{4}-4x^{2}+3$$

となります。$2x^{4}-4x^{2}$ を因数分解すると、

$$2x^{4}-4x^{2} = 2x^{2}(x^{2}-2)$$

となります。$|x|$ が小さいとき、$x^{2} < 2$ すなわち $x^{2}-2 < 0$ であるため、$2x^{2}(x^{2}-2) < 0$ です。したがって、

$$f(x,x)-3 = 2x^{4}-4x^{2} = 2x^{2}(x^{2}-2) < 0$$

すなわち、$f(x,x) < 3 = f(0,0)$ です。

したがって、$(0,0)$ の近傍に $f(x,y) > f(0,0)$ となる点（$y=-x$ 経路）と $f(x,y) < f(0,0)$ となる点（$y=x$ 経路）が存在するため、$(0,0)$ は極値ではありません。

$(1,1)$：$D(1,1)=24(-6+1+1)=-96<0$。$f_{xx}(1,1)=12(1)^{2}-2=10>0$ より極小。値は
$$f(1,1)=1+1-1-2-1+3=1$$
$(-1,-1)$：$D(-1,-1)=-96<0$。$f_{xx}(-1,-1)=12(-1)^{2}-2=10>0$ より極小。値は
$$f(-1,-1)=1+1-1-2-1+3=1$$
補足： $D<0$ のときはヘッセ行列の符号判定で、$f_{xx}>0$ なら極小、$f_{xx}<0$ なら極大です。$D=0$ のときは定義に戻って近傍の値を比較する必要があります。
【検証】

偏導関数と2階偏導関数の計算：$f_{x}=4x^{3}-2x-2y$、$f_{y}=4y^{3}-2x-2y$、$f_{xx}=12x^{2}-2$、$f_{yy}=12y^{2}-2$、$f_{xy}=-2$ は正しいです。

停留点の求解：$x^{3}-y^{3}=0$ から $x=y$、代入で $x=0,\pm1$ を得て $(0,0),(1,1),(-1,-1)$ は妥当です。

判別式：$D=24(-6x^{2}y^{2}+x^{2}+y^{2})$ は展開結果と一致します。$(1,1),(-1,-1)$ で $D<0$ かつ $f_{xx}>0$ より極小、値はいずれも $1$ で計算と整合します。
