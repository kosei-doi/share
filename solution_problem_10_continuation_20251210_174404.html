微分積分 10 解答

【問題】

次の関数の極値を調べよ。

（2）$f(x,y)=x^{3}+y^{3}-3xy$

（3）$f(x,y)=x^{4}+y^{4}$

（4）$f(x,y)=2x^{4}-3x^{2}y+y^{2}$

【方針】
必要な知識・テクニック
  • 偏微分：2変数関数 $f(x,y)$ の $x$ に関する偏微分 $f_{x}(x,y)$ は、$y$ を定数として $x$ で微分したものです。同様に、$y$ に関する偏微分 $f_{y}(x,y)$ は、$x$ を定数として $y$ で微分したものです。
  • 停留点：$f_{x}(x,y) = 0$ かつ $f_{y}(x,y) = 0$ を満たす点 $(x,y)$ を停留点といいます。極値は停留点でのみ現れる可能性がありますが、すべての停留点が極値であるとは限りません。
  • 2階偏導関数：$f_{xx} = \frac{\partial^{2}f}{\partial x^{2}}$、$f_{yy} = \frac{\partial^{2}f}{\partial y^{2}}$、$f_{xy} = \frac{\partial^{2}f}{\partial x \partial y}$ を2階偏導関数といいます。$f_{xy} = f_{yx}$ が成り立ちます（連続な場合）。
  • ヘッセ行列の判別式：$D = (f_{xy})^{2} - f_{xx}f_{yy}$ を判別式といいます。停留点 $(a,b)$ において、$D(a,b) < 0$ のとき極値の可能性があり、$D(a,b) > 0$ のとき極値ではありません（鞍点）。$D(a,b) = 0$ のときは判別式だけでは判定できません。
  • 極値の判定：停留点 $(a,b)$ において、$D(a,b) < 0$ かつ $f_{xx}(a,b) > 0$ のとき極小、$D(a,b) < 0$ かつ $f_{xx}(a,b) < 0$ のとき極大となります。
  • 判別式が0の場合の判定：判別式が0の場合は、判別式だけでは判定できません。極値の定義に基づいて、$(a,b)$ の近傍で $f(x,y) \ge f(a,b)$（極小）または $f(x,y) \le f(a,b)$（極大）が成り立つか、または異なる経路で $(a,b)$ に近づけたときの関数値の変化を調べます。
開始
各関数の極値を調べる
ステップ1
1階偏導関数 $f_{x}$、$f_{y}$ を計算する
ステップ2
$f_{x} = 0$、$f_{y} = 0$ を解き、停留点を求める
ステップ3
2階偏導関数 $f_{xx}$、$f_{yy}$、$f_{xy}$ を計算する
ステップ4
判別式 $D = (f_{xy})^{2} - f_{xx}f_{yy}$ を計算する
判定
各停留点で判別式の符号を調べる
$D > 0$ の場合
極値でない（鞍点）
$D < 0$ の場合
$f_{xx}$ の符号から極大・極小を判定する
$D = 0$ の場合
極値の定義に基づいて個別に調べる
終了
各関数の極値の結果をまとめる

【解答】

この問題では、3つの関数について、まず1階偏導関数を計算して停留点を求め、次に2階偏導関数とヘッセ行列の判別式を用いて各停留点での極値の判定を行います。判別式が正の場合は鞍点、負の場合は $f_{xx}$ の符号から極大・極小を判定します。判別式が0の場合は、極値の定義に基づいて個別に調べます。

（2）関数 $f(x,y)=x^{3}+y^{3}-3xy$ について

まず、1階偏導関数を計算します。$f(x,y) = x^{3} + y^{3} - 3xy$ について、$x$ に関する偏微分 $f_{x}(x,y)$ を計算します。$y$ を定数として $x$ で微分します。

$x^{3}$ を $x$ で偏微分すると $3x^{2}$ です。$y^{3}$ は $x$ を含まないため、$x$ で偏微分すると $0$ です。$-3xy$ を $x$ で偏微分すると $-3y$ です。したがって、

$$f_{x}(x,y) = 3x^{2} - 3y$$

となります。

次に、$y$ に関する偏微分 $f_{y}(x,y)$ を計算します。$x$ を定数として $y$ で微分します。

$x^{3}$ は $y$ を含まないため、$y$ で偏微分すると $0$ です。$y^{3}$ を $y$ で偏微分すると $3y^{2}$ です。$-3xy$ を $y$ で偏微分すると $-3x$ です。したがって、

$$f_{y}(x,y) = 3y^{2} - 3x$$

となります。

次に、停留点を求めます。停留点は $f_{x}(x,y) = 0$ かつ $f_{y}(x,y) = 0$ を満たす点です。したがって、連立方程式

$$\begin{cases} f_{x}(x,y) = 3x^{2} - 3y = 0 \\ f_{y}(x,y) = 3y^{2} - 3x = 0 \end{cases}$$

を解きます。

まず、$f_{x}(x,y) = 0$ より、

$$3x^{2} - 3y = 0$$

となります。両辺を $3$ で割ると、

$$x^{2} - y = 0$$

すなわち、

$$y = x^{2}$$

となります。

次に、$f_{y}(x,y) = 0$ より、

$$3y^{2} - 3x = 0$$

となります。両辺を $3$ で割ると、

$$y^{2} - x = 0$$

すなわち、

$$x = y^{2}$$

となります。

したがって、$y = x^{2}$ と $x = y^{2}$ を同時に満たす点を求めます。$x = y^{2}$ に $y = x^{2}$ を代入すると、

$$x = (x^{2})^{2} = x^{4}$$

となります。したがって、

$$x^{4} - x = 0$$

となります。左辺を因数分解すると、

$$x(x^{3} - 1) = 0$$

となります。したがって、$x = 0$ または $x^{3} - 1 = 0$ すなわち $x^{3} = 1$ です。$x^{3} = 1$ の実数解は $x = 1$ です。

補足： $x^{3} - 1 = 0$ は $x^{3} = 1$ となり、実数解は $x = 1$ のみです。複素数解もありますが、ここでは実数解のみを考えます。

$x = 0$ のとき、$y = x^{2} = 0^{2} = 0$ です。$x = 1$ のとき、$y = x^{2} = 1^{2} = 1$ です。

したがって、極値候補（停留点）は、$(0,0)$、$(1,1)$ です。

次に、2階偏導関数を計算します。まず、$f_{xx}(x,y)$ を計算します。$f_{x}(x,y) = 3x^{2} - 3y$ を $x$ で偏微分すると、

$$f_{xx}(x,y) = \frac{\partial}{\partial x}(3x^{2} - 3y) = 6x - 0 = 6x$$

となります。ここで、$3x^{2}$ を $x$ で偏微分すると $6x$ です。$-3y$ は $x$ を含まないため、$x$ で偏微分すると $0$ です。

次に、$f_{yy}(x,y)$ を計算します。$f_{y}(x,y) = 3y^{2} - 3x$ を $y$ で偏微分すると、

$$f_{yy}(x,y) = \frac{\partial}{\partial y}(3y^{2} - 3x) = 6y - 0 = 6y$$

となります。ここで、$3y^{2}$ を $y$ で偏微分すると $6y$ です。$-3x$ は $y$ を含まないため、$y$ で偏微分すると $0$ です。

次に、$f_{xy}(x,y)$ を計算します。$f_{x}(x,y) = 3x^{2} - 3y$ を $y$ で偏微分すると、

$$f_{xy}(x,y) = \frac{\partial}{\partial y}(3x^{2} - 3y) = 0 - 3 = -3$$

となります。ここで、$3x^{2}$ は $y$ を含まないため、$y$ で偏微分すると $0$ です。$-3y$ を $y$ で偏微分すると $-3$ です。

次に、ヘッセ行列の判別式 $D$ を計算します。判別式は、

$$D = (f_{xy})^{2} - f_{xx}f_{yy}$$

で定義されます。$f_{xx} = 6x$、$f_{yy} = 6y$、$f_{xy} = -3$ であるため、

$$D = (-3)^{2} - (6x)(6y) = 9 - 36xy$$

となります。ここで、$(-3)^{2} = 9$、$(6x)(6y) = 36xy$ であることを用いました。

各停留点での判別式の値と極値の判定を行います。

まず、$(0,0)$ について、

$$D(0,0) = 9 - 36 \cdot 0 \cdot 0 = 9 - 0 = 9 > 0$$

となります。判別式が正であるため、$(0,0)$ は極値ではありません。この点は鞍点（saddle point）です。

補足： 鞍点とは、ある方向では極大、別の方向では極小となる点です。判別式が正の場合は、その停留点は極値ではありません。

次に、$(1,1)$ について、

$$D(1,1) = 9 - 36 \cdot 1 \cdot 1 = 9 - 36 = -27 < 0$$

となります。判別式が負であるため、$(1,1)$ は極値の可能性があります。次に、$f_{xx}(1,1)$ の符号を調べます。

$$f_{xx}(1,1) = 6 \cdot 1 = 6 > 0$$

となります。$f_{xx}(1,1) > 0$ であるため、$(1,1)$ は極小です。

補足： 判別式が負で $f_{xx} > 0$ のとき極小、$f_{xx} < 0$ のとき極大となります。これは、2次形式の符号から判定できます。

極小値は、

$$f(1,1) = 1^{3} + 1^{3} - 3(1)(1) = 1 + 1 - 3 = -1$$

となります。

（3）関数 $f(x,y)=x^{4}+y^{4}$ について

まず、1階偏導関数を計算します。$f(x,y) = x^{4} + y^{4}$ について、$x$ に関する偏微分 $f_{x}(x,y)$ を計算します。$y$ を定数として $x$ で微分します。

$x^{4}$ を $x$ で偏微分すると $4x^{3}$ です。$y^{4}$ は $x$ を含まないため、$x$ で偏微分すると $0$ です。したがって、

$$f_{x}(x,y) = 4x^{3}$$

となります。

次に、$y$ に関する偏微分 $f_{y}(x,y)$ を計算します。$x$ を定数として $y$ で微分します。

$x^{4}$ は $y$ を含まないため、$y$ で偏微分すると $0$ です。$y^{4}$ を $y$ で偏微分すると $4y^{3}$ です。したがって、

$$f_{y}(x,y) = 4y^{3}$$

となります。

次に、停留点を求めます。停留点は $f_{x}(x,y) = 0$ かつ $f_{y}(x,y) = 0$ を満たす点です。したがって、

$$f_{x}(x,y) = 4x^{3} = 0$$

および

$$f_{y}(x,y) = 4y^{3} = 0$$

を解きます。$4x^{3} = 0$ より $x = 0$、$4y^{3} = 0$ より $y = 0$ です。

したがって、極値候補（停留点）は、$(0,0)$ のみです。

次に、2階偏導関数を計算します。まず、$f_{xx}(x,y)$ を計算します。$f_{x}(x,y) = 4x^{3}$ を $x$ で偏微分すると、

$$f_{xx}(x,y) = \frac{\partial}{\partial x}(4x^{3}) = 12x^{2}$$

となります。ここで、$4x^{3}$ を $x$ で偏微分すると $12x^{2}$ です。

次に、$f_{yy}(x,y)$ を計算します。$f_{y}(x,y) = 4y^{3}$ を $y$ で偏微分すると、

$$f_{yy}(x,y) = \frac{\partial}{\partial y}(4y^{3}) = 12y^{2}$$

となります。ここで、$4y^{3}$ を $y$ で偏微分すると $12y^{2}$ です。

次に、$f_{xy}(x,y)$ を計算します。$f_{x}(x,y) = 4x^{3}$ を $y$ で偏微分すると、

$$f_{xy}(x,y) = \frac{\partial}{\partial y}(4x^{3}) = 0$$

となります。$4x^{3}$ は $y$ を含まないため、$y$ で偏微分すると $0$ です。

次に、ヘッセ行列の判別式 $D$ を計算します。$f_{xx} = 12x^{2}$、$f_{yy} = 12y^{2}$、$f_{xy} = 0$ であるため、

$$D = 0^{2} - (12x^{2})(12y^{2}) = -144x^{2}y^{2}$$

となります。ここで、$0^{2} = 0$、$(12x^{2})(12y^{2}) = 144x^{2}y^{2}$ であることを用いました。

停留点 $(0,0)$ での判別式の値を計算します。

$$D(0,0) = -144 \cdot 0^{2} \cdot 0^{2} = -144 \cdot 0 = 0$$

となります。判別式が $0$ であるため、判別式だけでは極値の判定ができません。極値の定義に基づいて調べます。

補足： 判別式が $0$ の場合は、判別式だけでは極値かどうかを判定できません。極値の定義に基づいて、$(0,0)$ の近傍で関数値の変化を調べる必要があります。

まず、$f(0,0) = 0^{4} + 0^{4} = 0$ です。

次に、任意の $(x,y) \in \mathbb{R}^{2}$ について、$f(x,y) = x^{4} + y^{4}$ を考えます。$x^{4} \ge 0$、$y^{4} \ge 0$ であるため、

$$f(x,y) = x^{4} + y^{4} \ge 0$$

が成り立ちます。したがって、

$$f(0,0) = 0 \le f(x,y)$$

がすべての $(x,y) \in \mathbb{R}^{2}$ に対して成り立ちます。これは、$(0,0)$ の近傍で $f(x,y) \ge f(0,0)$ が成り立つことを意味します。

補足： 極小の定義は、$(a,b)$ の近傍で $f(x,y) \ge f(a,b)$ が成り立つことです。ここでは、すべての $(x,y)$ に対して $f(x,y) \ge f(0,0) = 0$ が成り立つため、$(0,0)$ は極小です。

したがって、$(0,0)$ は極小で、極小値は $f(0,0) = 0$ です。

（4）関数 $f(x,y)=2x^{4}-3x^{2}y+y^{2}$ について

まず、1階偏導関数を計算します。$f(x,y) = 2x^{4} - 3x^{2}y + y^{2}$ について、$x$ に関する偏微分 $f_{x}(x,y)$ を計算します。$y$ を定数として $x$ で微分します。

$2x^{4}$ を $x$ で偏微分すると $8x^{3}$ です。$-3x^{2}y$ を $x$ で偏微分すると $-6xy$ です。$y^{2}$ は $x$ を含まないため、$x$ で偏微分すると $0$ です。したがって、

$$f_{x}(x,y) = 8x^{3} - 6xy$$

となります。

次に、$y$ に関する偏微分 $f_{y}(x,y)$ を計算します。$x$ を定数として $y$ で微分します。

$2x^{4}$ は $y$ を含まないため、$y$ で偏微分すると $0$ です。$-3x^{2}y$ を $y$ で偏微分すると $-3x^{2}$ です。$y^{2}$ を $y$ で偏微分すると $2y$ です。したがって、

$$f_{y}(x,y) = -3x^{2} + 2y$$

となります。

次に、停留点を求めます。停留点は $f_{x}(x,y) = 0$ かつ $f_{y}(x,y) = 0$ を満たす点です。したがって、連立方程式

$$\begin{cases} f_{x}(x,y) = 8x^{3} - 6xy = 0 \\ f_{y}(x,y) = -3x^{2} + 2y = 0 \end{cases}$$

を解きます。

まず、$f_{y}(x,y) = 0$ より、

$$-3x^{2} + 2y = 0$$

となります。したがって、

$$2y = 3x^{2}$$

すなわち、

$$y = \frac{3}{2}x^{2}$$

となります。

次に、$f_{x}(x,y) = 0$ に $y = \frac{3}{2}x^{2}$ を代入します。したがって、

$$8x^{3} - 6x \cdot \frac{3}{2}x^{2} = 0$$

となります。計算すると、

$$8x^{3} - 6x \cdot \frac{3}{2}x^{2} = 8x^{3} - 9x^{3} = -x^{3} = 0$$

となります。ここで、$6 \cdot \frac{3}{2} = 9$、$x \cdot x^{2} = x^{3}$ であることを用いました。したがって、$x = 0$ です。

$x = 0$ のとき、$y = \frac{3}{2} \cdot 0^{2} = 0$ です。

したがって、極値候補（停留点）は、$(0,0)$ のみです。

次に、2階偏導関数を計算します。まず、$f_{xx}(x,y)$ を計算します。$f_{x}(x,y) = 8x^{3} - 6xy$ を $x$ で偏微分すると、

$$f_{xx}(x,y) = \frac{\partial}{\partial x}(8x^{3} - 6xy) = 24x^{2} - 6y$$

となります。ここで、$8x^{3}$ を $x$ で偏微分すると $24x^{2}$ です。$-6xy$ を $x$ で偏微分すると $-6y$ です。

次に、$f_{yy}(x,y)$ を計算します。$f_{y}(x,y) = -3x^{2} + 2y$ を $y$ で偏微分すると、

$$f_{yy}(x,y) = \frac{\partial}{\partial y}(-3x^{2} + 2y) = 0 + 2 = 2$$

となります。ここで、$-3x^{2}$ は $y$ を含まないため、$y$ で偏微分すると $0$ です。$2y$ を $y$ で偏微分すると $2$ です。

次に、$f_{xy}(x,y)$ を計算します。$f_{x}(x,y) = 8x^{3} - 6xy$ を $y$ で偏微分すると、

$$f_{xy}(x,y) = \frac{\partial}{\partial y}(8x^{3} - 6xy) = 0 - 6x = -6x$$

となります。ここで、$8x^{3}$ は $y$ を含まないため、$y$ で偏微分すると $0$ です。$-6xy$ を $y$ で偏微分すると $-6x$ です。

次に、ヘッセ行列の判別式 $D$ を計算します。$f_{xx} = 24x^{2} - 6y$、$f_{yy} = 2$、$f_{xy} = -6x$ であるため、

$$D = (-6x)^{2} - (24x^{2} - 6y)(2) = 36x^{2} - 48x^{2} + 12y = -12x^{2} + 12y$$

となります。ここで、$(-6x)^{2} = 36x^{2}$、$(24x^{2} - 6y)(2) = 48x^{2} - 12y$ であることを用いました。したがって、$D = 36x^{2} - (48x^{2} - 12y) = 36x^{2} - 48x^{2} + 12y = -12x^{2} + 12y$ となります。

停留点 $(0,0)$ での判別式の値を計算します。

$$D(0,0) = -12 \cdot 0^{2} + 12 \cdot 0 = 0 + 0 = 0$$

となります。判別式が $0$ であるため、判別式だけでは極値の判定ができません。極値の定義に基づいて調べます。

補足： 判別式が $0$ の場合は、判別式だけでは極値かどうかを判定できません。異なる経路で $(0,0)$ に近づけたときの関数値の変化を調べる必要があります。

まず、$f(0,0) = 2(0)^{4} - 3(0)^{2}(0) + (0)^{2} = 0$ です。

次に、異なる経路で $(0,0)$ に近づけたときの関数値の変化を調べます。$y = mx^{2}$（$m$ は定数）という経路で $(0,0)$ に近づける場合を考えます。このとき、$y = mx^{2}$ を代入すると、

$$f(x,mx^{2}) = 2x^{4} - 3x^{2}(mx^{2}) + (mx^{2})^{2} = 2x^{4} - 3mx^{4} + m^{2}x^{4} = x^{4}(2 - 3m + m^{2})$$

となります。ここで、$-3x^{2}(mx^{2}) = -3mx^{4}$、$(mx^{2})^{2} = m^{2}x^{4}$ であることを用いました。

$2 - 3m + m^{2}$ を因数分解すると、

$$2 - 3m + m^{2} = m^{2} - 3m + 2 = (m-1)(m-2)$$

となります。ここで、$m^{2} - 3m + 2 = (m-1)(m-2)$ という因数分解公式を用いました。

補足： $m^{2} - 3m + 2$ の因数分解は、$m = 1$ のとき $1 - 3 + 2 = 0$、$m = 2$ のとき $4 - 6 + 2 = 0$ であることから、$(m-1)(m-2)$ と因数分解できます。

したがって、

$$f(x,mx^{2}) = x^{4}(m-1)(m-2)$$

となります。

ここで、$m$ の値によって $f(x,mx^{2})$ の符号が異なることを確認します。

例えば、$m = 3$ のとき、

$$f(x,3x^{2}) = x^{4}(3-1)(3-2) = x^{4} \cdot 2 \cdot 1 = 2x^{4} > 0 = f(0,0)$$

となります。ここで、$x \ne 0$ のとき $x^{4} > 0$ であるため、$f(x,3x^{2}) > 0 = f(0,0)$ です。

また、$m = 0$ のとき、

$$f(x,0) = 2x^{4} - 3x^{2}(0) + 0^{2} = 2x^{4} > 0 = f(0,0)$$

となります。$x \ne 0$ のとき $x^{4} > 0$ であるため、$f(x,0) > 0 = f(0,0)$ です。

一方、$m = \frac{3}{2}$ のとき、

$$f(x,\frac{3}{2}x^{2}) = x^{4}\left(\frac{3}{2}-1\right)\left(\frac{3}{2}-2\right) = x^{4} \cdot \frac{1}{2} \cdot \left(-\frac{1}{2}\right) = -\frac{1}{4}x^{4} < 0 = f(0,0)$$

となります。ここで、$\frac{3}{2} - 1 = \frac{1}{2}$、$\frac{3}{2} - 2 = -\frac{1}{2}$ であることを用いました。$x \ne 0$ のとき $x^{4} > 0$ であるため、$f(x,\frac{3}{2}x^{2}) < 0 = f(0,0)$ です。

補足： 極値の定義では、$(a,b)$ の近傍で $f(x,y) \ge f(a,b)$（極小）または $f(x,y) \le f(a,b)$（極大）が成り立つ必要があります。ここでは、$m = 3$ や $m = 0$ の経路では $f(x,mx^{2}) > f(0,0)$ となり、$m = \frac{3}{2}$ の経路では $f(x,mx^{2}) < f(0,0)$ となります。したがって、$(0,0)$ の近傍で、$f(0,0)$ より大きい値も小さい値も存在するため、$(0,0)$ は極値ではありません。

したがって、$(0,0)$ は極値ではありません。異なる経路で $(0,0)$ に近づけたとき、関数値が $f(0,0) = 0$ より大きくなる場合も小さくなる場合も存在するためです。

（2）$f(x,y)=x^{3}+y^{3}-3xy$ について：

停留点は $(0,0)$、$(1,1)$ です。

$(0,0)$ は極値ではありません（鞍点）。判別式 $D(0,0) = 9 > 0$ であるためです。

$(1,1)$ は極小で、極小値は $f(1,1) = -1$ です。判別式 $D(1,1) = -27 < 0$ かつ $f_{xx}(1,1) = 6 > 0$ であるためです。

（3）$f(x,y)=x^{4}+y^{4}$ について：

停留点は $(0,0)$ のみです。

$(0,0)$ は極小で、極小値は $f(0,0) = 0$ です。判別式 $D(0,0) = 0$ であるため判別式だけでは判定できませんが、すべての $(x,y) \in \mathbb{R}^{2}$ に対して $f(x,y) = x^{4} + y^{4} \ge 0 = f(0,0)$ が成り立つためです。

（4）$f(x,y)=2x^{4}-3x^{2}y+y^{2}$ について：

停留点は $(0,0)$ のみです。

$(0,0)$ は極値ではありません。判別式 $D(0,0) = 0$ であるため判別式だけでは判定できませんが、$y = mx^{2}$ という経路で $(0,0)$ に近づけると、$m = 3$ や $m = 0$ のとき $f(x,mx^{2}) > 0 = f(0,0)$、$m = \frac{3}{2}$ のとき $f(x,mx^{2}) < 0 = f(0,0)$ となるためです。
