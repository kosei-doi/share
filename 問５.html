微分積分 問5 解答

【問題】

関数 $f(\boldsymbol{x})$ の点 $(1,2)$ における $\boldsymbol{a} = (\frac{1}{\sqrt{2}},-\frac{1}{\sqrt{2}})$ 方向への微分の定義を書け。

【方針】
必要な知識・テクニック
  • 方向微分：多変数関数 $f(\boldsymbol{x})$ において、特定の方向 $\boldsymbol{a}$ への変化率を表す概念です。点 $\boldsymbol{x}_0$ における方向 $\boldsymbol{a}$ への方向微分は、$\boldsymbol{x}_0$ から $\boldsymbol{a}$ 方向に微小距離 $t$ だけ移動したときの関数値の変化率として定義されます。
  • 方向ベクトル：方向微分を定義する際には、方向を表すベクトル $\boldsymbol{a}$ が必要です。通常、方向ベクトルは単位ベクトル（長さが1のベクトル）として扱われます。この問題では、$\boldsymbol{a} = (\frac{1}{\sqrt{2}},-\frac{1}{\sqrt{2}})$ は単位ベクトルです（$\|\boldsymbol{a}\| = \sqrt{(\frac{1}{\sqrt{2}})^2 + (-\frac{1}{\sqrt{2}})^2} = 1$）。
  • 極限の定義：方向微分は極限として定義されます。$t \to 0$ のときの極限を考えることで、点 $\boldsymbol{x}_0$ における瞬間的な変化率を表します。これは1変数関数の微分の定義 $f'(x) = \lim_{h \to 0} \frac{f(x+h) - f(x)}{h}$ の多変数への拡張です。
  • ベクトルの和とスカラー倍：方向微分の定義では、点 $\boldsymbol{x}_0$ に方向ベクトル $\boldsymbol{a}$ の $t$ 倍を加えた点 $\boldsymbol{x}_0 + t\boldsymbol{a}$ を考えます。これは、$\boldsymbol{x}_0$ から $\boldsymbol{a}$ 方向に距離 $|t|$ だけ移動した点を表します。

【解答】

この問題では、方向微分の一般的な定義を確認し、与えられた点 $(1,2)$ と方向ベクトル $\boldsymbol{a} = (\frac{1}{\sqrt{2}},-\frac{1}{\sqrt{2}})$ を具体的な定義式に代入します。まず、方向微分の概念を説明し、次に具体的な計算を行って定義式を完成させます。

（1）方向微分の一般的な定義

多変数関数 $f(\boldsymbol{x})$ の点 $\boldsymbol{x}_0$ における方向ベクトル $\boldsymbol{a}$ への方向微分は、次の極限として定義されます。

$$\lim_{t \to 0} \frac{f(\boldsymbol{x}_0 + t\boldsymbol{a}) - f(\boldsymbol{x}_0)}{t}$$

ここで、$t$ は実数で、$t \to 0$ の極限を考えます。$\boldsymbol{x}_0 + t\boldsymbol{a}$ は、点 $\boldsymbol{x}_0$ から方向ベクトル $\boldsymbol{a}$ に沿って距離 $|t|$ だけ移動した点を表します。この定義は、1変数関数の微分の定義 $f'(x) = \lim_{h \to 0} \frac{f(x+h) - f(x)}{h}$ の多変数への拡張です。

方向微分は、関数が特定の方向にどれだけ変化するかを表します。偏微分は座標軸方向（$x$ 方向や $y$ 方向）への方向微分の特別な場合です。方向微分により、任意の方向への関数の変化率を考えることができます。
（2）与えられた条件の確認

問題では、点 $\boldsymbol{x}_0 = (1, 2)$ と方向ベクトル $\boldsymbol{a} = (\frac{1}{\sqrt{2}},-\frac{1}{\sqrt{2}})$ が与えられています。

まず、方向ベクトル $\boldsymbol{a}$ が単位ベクトルであることを確認します。ベクトルの長さ（ノルム）は、

$$\|\boldsymbol{a}\| = \sqrt{\left(\frac{1}{\sqrt{2}}\right)^2 + \left(-\frac{1}{\sqrt{2}}\right)^2} = \sqrt{\frac{1}{2} + \frac{1}{2}} = \sqrt{1} = 1$$

となり、確かに単位ベクトルです。方向ベクトルは通常単位ベクトルとして扱われますが、この問題では既に単位ベクトルとして与えられています。

次に、点 $(1,2)$ から方向 $\boldsymbol{a}$ に $t$ だけ移動した点を計算します。

$$\boldsymbol{x}_0 + t\boldsymbol{a} = (1, 2) + t\left(\frac{1}{\sqrt{2}},-\frac{1}{\sqrt{2}}\right) = \left(1 + \frac{t}{\sqrt{2}}, 2 - \frac{t}{\sqrt{2}}\right)$$

この点の座標は、$(1+\frac{t}{\sqrt{2}}, 2-\frac{t}{\sqrt{2}})$ となります。

（3）方向微分の定義式への代入

方向微分の一般的な定義式に、具体的な値を代入します。

$\boldsymbol{x}_0 = (1, 2)$、$\boldsymbol{a} = (\frac{1}{\sqrt{2}},-\frac{1}{\sqrt{2}})$ より、$\boldsymbol{x}_0 + t\boldsymbol{a} = (1+\frac{t}{\sqrt{2}}, 2-\frac{t}{\sqrt{2}})$ となります。

したがって、方向微分の定義式は、

$$\lim_{t \to 0} \frac{f(1+\frac{t}{\sqrt{2}}, 2-\frac{t}{\sqrt{2}}) - f(1,2)}{t}$$

となります。

この式は、点 $(1,2)$ から方向 $(\frac{1}{\sqrt{2}},-\frac{1}{\sqrt{2}})$ に微小距離 $t$ だけ移動したときの関数値の変化率を表しています。分子は関数値の変化量、分母は移動距離を表し、その比の極限が方向微分となります。

方向ベクトル $\boldsymbol{a} = (\frac{1}{\sqrt{2}},-\frac{1}{\sqrt{2}})$ は、$x$ 軸方向に $\frac{1}{\sqrt{2}}$、$y$ 軸方向に $-\frac{1}{\sqrt{2}}$ の成分を持つベクトルです。これは、$x$ 方向に正の成分、$y$ 方向に負の成分を持つ方向を表しています。
（4）定義式の完成

以上より、関数 $f(\boldsymbol{x})$ の点 $(1,2)$ における $\boldsymbol{a} = (\frac{1}{\sqrt{2}},-\frac{1}{\sqrt{2}})$ 方向への微分の定義は、

$$\lim_{t \to 0} \frac{f(1+\frac{t}{\sqrt{2}}, 2-\frac{t}{\sqrt{2}}) - f(1,2)}{t}$$

で与えられます。

この定義式は、次のように解釈できます。点 $(1,2)$ から方向ベクトル $\boldsymbol{a} = (\frac{1}{\sqrt{2}},-\frac{1}{\sqrt{2}})$ に沿って微小距離 $t$ だけ移動した点 $(1+\frac{t}{\sqrt{2}}, 2-\frac{t}{\sqrt{2}})$ における関数値と、元の点 $(1,2)$ における関数値の差を、移動距離 $t$ で割った値の極限（$t \to 0$）が方向微分です。

方向微分が存在するためには、この極限が存在する必要があります。関数が全微分可能である場合、方向微分は偏導関数と方向ベクトルの内積として表すことができ、$\frac{\partial f}{\partial \boldsymbol{a}} = \nabla f \cdot \boldsymbol{a} = f_x \boldsymbol{a}_x + f_y \boldsymbol{a}_y$ となります。ただし、この問題では定義を書くことが求められているため、極限による定義式を書きます。
【検証】

まず、方向ベクトル $\boldsymbol{a} = (\frac{1}{\sqrt{2}},-\frac{1}{\sqrt{2}})$ が単位ベクトルであることを確認します。$\|\boldsymbol{a}\| = \sqrt{(\frac{1}{\sqrt{2}})^2 + (-\frac{1}{\sqrt{2}})^2} = \sqrt{\frac{1}{2} + \frac{1}{2}} = 1$ となり、確かに単位ベクトルです。方向微分の定義では、方向ベクトルは通常単位ベクトルとして扱われるため、この確認は重要です。

次に、点 $(1,2)$ から方向 $\boldsymbol{a}$ に $t$ だけ移動した点が正しく計算されることを確認します。$\boldsymbol{x}_0 + t\boldsymbol{a} = (1, 2) + t(\frac{1}{\sqrt{2}},-\frac{1}{\sqrt{2}}) = (1 + \frac{t}{\sqrt{2}}, 2 - \frac{t}{\sqrt{2}})$ となり、$t = 0$ のとき $(1,2)$ に戻り、$t > 0$ のとき $x$ 座標は増加し、$y$ 座標は減少することが確認できます。これは方向ベクトル $\boldsymbol{a}$ の成分が正と負であることと一致しています。

最後に、定義式の構造が正しいことを確認します。分子は関数値の変化量 $f(\boldsymbol{x}_0 + t\boldsymbol{a}) - f(\boldsymbol{x}_0)$ を表し、分母は移動距離 $t$ を表します。この比の極限が方向微分であり、1変数関数の微分の定義と同様の構造を持っていることを確認できます。

以上より求めた結果の正しさを確認しました。
