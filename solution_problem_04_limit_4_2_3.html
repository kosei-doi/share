微分積分 04 解答

【問題】

次の極限を求めよ。

（1）$\lim_{(x,y)\to(0,0)} \frac{xy}{\sqrt{x^{2}+y^{2}}}$

（2）$\lim_{(x,y)\to(0,0)} \frac{xy}{x^{2}+y^{2}}$

（3）$\lim_{(x,y)\to(0,0)} \frac{x^{3}+y^{3}}{x^{2}+y^{2}}$

（4）$\lim_{(x,y)\to(0,0)} \frac{xy^{2}}{x^{2}+y^{4}}$

【方針】
必要な知識・テクニック
  • 2変数関数の極限：2変数関数 $f(x,y)$ の $(x,y) \to (0,0)$ における極限は、$(x,y)$ が $(0,0)$ に近づくすべての経路で同じ値に収束する必要があります。1つの経路で異なる値になる、または発散する場合は極限は存在しません。
  • 極座標変換：$x = r\cos\theta$、$y = r\sin\theta$ とおくことで、$(x,y) \to (0,0)$ は $r \to 0$ に対応します。この変換により、極限の計算が簡潔になる場合があります。
  • はさみうちの原理：関数 $f(x,y)$ に対して、$|f(x,y)| \le g(r)$ かつ $\lim_{r \to 0} g(r) = 0$ が成り立つとき、$\lim_{(x,y)\to(0,0)} f(x,y) = 0$ となります。これは極限が存在することを示す強力な方法です。
  • 経路依存性による極限の非存在の証明：異なる経路（例えば、$y=kx$ や $y^{2}=kx$ など）で $(0,0)$ に近づけたとき、極限値が異なる値になることを示すことで、極限が存在しないことを証明できます。
  • 三角関数の性質：$\cos^{2}\theta + \sin^{2}\theta = 1$ という恒等式を利用して、極座標変換後の式を簡約します。
開始
各極限について、極座標変換または経路依存性を調べる
問題（1）
極座標変換を行い、はさみうちの原理を適用する
問題（2）
極座標変換を行い、経路依存性を調べる
問題（3）
極座標変換を行い、はさみうちの原理を適用する
問題（4）
経路依存性を調べ、異なる経路で異なる極限値を示す
判定
極限が存在するか、経路依存で存在しないか
終了
各問題の結果をまとめる

【解答】

この問題では、4つの2変数関数の極限を計算します。問題（1）と（3）では、極座標変換とはさみうちの原理を用いて極限が存在することを示します。問題（2）と（4）では、異なる経路で $(0,0)$ に近づけたときの極限値が異なることを示し、極限が存在しないことを証明します。

（1）$\lim_{(x,y)\to(0,0)} \frac{xy}{\sqrt{x^{2}+y^{2}}}$ について

この極限を計算するために、極座標変換を用います。$x = r\cos\theta$、$y = r\sin\theta$ とおくと、$(x,y) \to (0,0)$ は $r \to 0$ に対応します。

まず、分子と分母を極座標で表します。分子は、

$$xy = r\cos\theta \cdot r\sin\theta = r^{2}\cos\theta\sin\theta$$

です。次に、分母は、

$$\sqrt{x^{2}+y^{2}} = \sqrt{r^{2}\cos^{2}\theta + r^{2}\sin^{2}\theta} = \sqrt{r^{2}(\cos^{2}\theta + \sin^{2}\theta)} = \sqrt{r^{2}} = |r| = r$$

ここで、$r \ge 0$ として考えているため、$\sqrt{r^{2}} = r$ となります。また、$\cos^{2}\theta + \sin^{2}\theta = 1$ という三角関数の恒等式を用いました。

したがって、極限は、

$$\lim_{(x,y)\to(0,0)} \frac{xy}{\sqrt{x^{2}+y^{2}}} = \lim_{r \to 0} \frac{r^{2}\cos\theta\sin\theta}{r} = \lim_{r \to 0} r\cos\theta\sin\theta$$

となります。ここで、$|\cos\theta\sin\theta| \le 1$ であることに注意します。したがって、

$$|r\cos\theta\sin\theta| \le |r| \cdot |\cos\theta\sin\theta| \le |r| = r$$

が成り立ちます。$g(r) = r$ とおくと、$|r\cos\theta\sin\theta| \le g(r)$ かつ $\lim_{r \to 0} g(r) = 0$ です。

補足： はさみうちの原理により、$|f(x,y)| \le g(r)$ かつ $\lim_{r \to 0} g(r) = 0$ が成り立つとき、$\lim_{(x,y)\to(0,0)} f(x,y) = 0$ となります。ここでは、$f(x,y) = r\cos\theta\sin\theta$ と見なしています。

したがって、はさみうちの原理より、

$$\lim_{(x,y)\to(0,0)} \frac{xy}{\sqrt{x^{2}+y^{2}}} = \lim_{r \to 0} r\cos\theta\sin\theta = 0$$

となります。

（2）$\lim_{(x,y)\to(0,0)} \frac{xy}{x^{2}+y^{2}}$ について

この極限を計算するために、極座標変換を用います。$x = r\cos\theta$、$y = r\sin\theta$ とおくと、

$$\lim_{(x,y)\to(0,0)} \frac{xy}{x^{2}+y^{2}} = \lim_{r \to 0} \frac{r^{2}\cos\theta\sin\theta}{r^{2}(\cos^{2}\theta + \sin^{2}\theta)} = \lim_{r \to 0} \frac{r^{2}\cos\theta\sin\theta}{r^{2}} = \lim_{r \to 0} \cos\theta\sin\theta$$

となります。ここで、$\cos^{2}\theta + \sin^{2}\theta = 1$ を用いました。

この極限は、$\theta$ の値に依存します。例えば、$\theta = \frac{\pi}{4}$ の方向から $(0,0)$ に近づけると、

$$\cos\frac{\pi}{4}\sin\frac{\pi}{4} = \frac{1}{\sqrt{2}} \cdot \frac{1}{\sqrt{2}} = \frac{1}{2}$$

となります。一方、$\theta = 0$ の方向（$x$ 軸に沿って）から $(0,0)$ に近づけると、$\cos 0 \sin 0 = 1 \cdot 0 = 0$ となります。

補足： 2変数関数の極限が存在するためには、$(x,y)$ が $(0,0)$ に近づくすべての経路で同じ値に収束する必要があります。ここでは、異なる方向（異なる $\theta$ の値）から近づけたときに異なる値になるため、極限は存在しません。

したがって、近づける方向によって値が変わるので、この極限は収束しません。すなわち、極限は存在しません。

（3）$\lim_{(x,y)\to(0,0)} \frac{x^{3}+y^{3}}{x^{2}+y^{2}}$ について

この極限を計算するために、極座標変換を用います。$x = r\cos\theta$、$y = r\sin\theta$ とおくと、

$$x^{3} + y^{3} = r^{3}\cos^{3}\theta + r^{3}\sin^{3}\theta = r^{3}(\cos^{3}\theta + \sin^{3}\theta)$$

であり、

$$x^{2} + y^{2} = r^{2}\cos^{2}\theta + r^{2}\sin^{2}\theta = r^{2}(\cos^{2}\theta + \sin^{2}\theta) = r^{2}$$

です。したがって、

$$\lim_{(x,y)\to(0,0)} \frac{x^{3}+y^{3}}{x^{2}+y^{2}} = \lim_{r \to 0} \frac{r^{3}(\cos^{3}\theta + \sin^{3}\theta)}{r^{2}} = \lim_{r \to 0} r(\cos^{3}\theta + \sin^{3}\theta)$$

となります。

ここで、$|\cos^{3}\theta + \sin^{3}\theta|$ の上界を求めます。$|\cos\theta| \le 1$、$|\sin\theta| \le 1$ であるため、

$$|\cos^{3}\theta + \sin^{3}\theta| \le |\cos^{3}\theta| + |\sin^{3}\theta| \le |\cos\theta|^{3} + |\sin\theta|^{3} \le 1 + 1 = 2$$

が成り立ちます。したがって、

$$|r(\cos^{3}\theta + \sin^{3}\theta)| = |r| \cdot |\cos^{3}\theta + \sin^{3}\theta| \le |r| \cdot 2 = 2r$$

となります。$g(r) = 2r$ とおくと、$|r(\cos^{3}\theta + \sin^{3}\theta)| \le g(r)$ かつ $\lim_{r \to 0} g(r) = 0$ です。

補足： はさみうちの原理を適用するために、$|f(x,y)| \le g(r)$ となる関数 $g(r)$ を見つける必要があります。ここでは、$|\cos^{3}\theta + \sin^{3}\theta| \le 2$ という上界を用いて、$g(r) = 2r$ としました。

したがって、はさみうちの原理より、

$$\lim_{(x,y)\to(0,0)} \frac{x^{3}+y^{3}}{x^{2}+y^{2}} = \lim_{r \to 0} r(\cos^{3}\theta + \sin^{3}\theta) = 0$$

となります。

（4）$\lim_{(x,y)\to(0,0)} \frac{xy^{2}}{x^{2}+y^{4}}$ について

この極限が存在するかどうかを調べるために、異なる経路で $(0,0)$ に近づけたときの極限値を計算します。

まず、$y = kx$（$k$ は定数）という経路で $(0,0)$ に近づける場合を考えます。このとき、$y = kx$ を代入すると、

$$\frac{xy^{2}}{x^{2}+y^{4}} = \frac{x(kx)^{2}}{x^{2}+(kx)^{4}} = \frac{k^{2}x^{3}}{x^{2}+k^{4}x^{4}} = \frac{k^{2}x^{3}}{x^{2}(1+k^{4}x^{2})} = \frac{k^{2}x}{1+k^{4}x^{2}}$$

となります。したがって、

$$\lim_{x \to 0} \frac{k^{2}x}{1+k^{4}x^{2}} = \frac{k^{2} \cdot 0}{1+k^{4} \cdot 0^{2}} = \frac{0}{1} = 0$$

となります。これはすべての $k$ に対して $0$ になるため、この経路だけでは極限の存在を判断できません。

より適切な経路として、$y^{2} = kx$（$k$ は定数）という関係を考えます。このとき、$y = \sqrt{kx}$ または $y = -\sqrt{kx}$（$x \ge 0$ のとき）となります。このとき、

$$\frac{xy^{2}}{x^{2}+y^{4}} = \frac{x \cdot kx}{x^{2}+(kx)^{2}} = \frac{kx^{2}}{x^{2}+k^{2}x^{2}} = \frac{kx^{2}}{x^{2}(1+k^{2})} = \frac{k}{1+k^{2}}$$

となります。したがって、$y^{2} = kx$ という経路で $(0,0)$ に近づけると、極限値は $\frac{k}{1+k^{2}}$ となります。

補足： $y^{2} = kx$ という関係は、$y = \sqrt{kx}$ または $y = -\sqrt{kx}$ と表せますが、$y^{2}$ が現れるため、どちらの符号でも同じ結果になります。この経路は、分母の $x^{2}$ と $y^{4}$ の次数の関係を考慮したものです。$y^{2} = kx$ とすると、$y^{4} = (kx)^{2} = k^{2}x^{2}$ となり、分母は $x^{2}+k^{2}x^{2} = x^{2}(1+k^{2})$ となります。

ここで、$k$ の値によって極限値が異なることを確認します。$k = 0$ のとき、$\frac{0}{1+0} = 0$ となります。$k = 1$ のとき、$\frac{1}{1+1} = \frac{1}{2}$ となります。

したがって、$k = 0$ の方向から近づけると極限値は $0$ となり、$k = 1$ の方向から近づけると極限値は $\frac{1}{2}$ となります。近づける方向によって値が変わるので、この極限は収束しません。すなわち、極限は存在しません。

【検証】

計算した内容を検証します。

まず、問題（1）の計算を確認します。極座標変換 $x = r\cos\theta$、$y = r\sin\theta$ を用いて、$\frac{xy}{\sqrt{x^{2}+y^{2}}} = r\cos\theta\sin\theta$ となりました。$|\cos\theta\sin\theta| \le 1$ であるため、$|r\cos\theta\sin\theta| \le r$ が成り立ちます。$g(r) = r$ とおくと、$\lim_{r \to 0} g(r) = 0$ であるため、はさみうちの原理より極限は $0$ となります。この計算は正しいです。

次に、問題（2）の計算を確認します。極座標変換により、$\frac{xy}{x^{2}+y^{2}} = \cos\theta\sin\theta$ となりました。これは $\theta$ に依存するため、異なる方向から $(0,0)$ に近づけると異なる値になります。例えば、$\theta = 0$ のとき $0$、$\theta = \frac{\pi}{4}$ のとき $\frac{1}{2}$ となります。したがって、極限は存在しません。この結論は正しいです。

問題（3）の計算を確認します。極座標変換により、$\frac{x^{3}+y^{3}}{x^{2}+y^{2}} = r(\cos^{3}\theta + \sin^{3}\theta)$ となりました。$|\cos^{3}\theta + \sin^{3}\theta| \le 2$ であるため、$|r(\cos^{3}\theta + \sin^{3}\theta)| \le 2r$ が成り立ちます。$g(r) = 2r$ とおくと、$\lim_{r \to 0} g(r) = 0$ であるため、はさみうちの原理より極限は $0$ となります。この計算は正しいです。

問題（4）の計算を確認します。$y = kx$ という経路では極限値は $0$ となりますが、$y^{2} = kx$ という経路では極限値は $\frac{k}{1+k^{2}}$ となります。$k = 0$ のとき $0$、$k = 1$ のとき $\frac{1}{2}$ となり、異なる経路で異なる極限値が得られます。したがって、極限は存在しません。この結論は正しいです。

【答】

（1）$\lim_{(x,y)\to(0,0)} \frac{xy}{\sqrt{x^{2}+y^{2}}} = 0$

極座標変換 $x = r\cos\theta$、$y = r\sin\theta$ を用いると、$\frac{xy}{\sqrt{x^{2}+y^{2}}} = r\cos\theta\sin\theta$ となります。$|r\cos\theta\sin\theta| \le r$ かつ $\lim_{r \to 0} r = 0$ であるため、はさみうちの原理より極限は $0$ となります。

（2）極限は存在しない

極座標変換により、$\frac{xy}{x^{2}+y^{2}} = \cos\theta\sin\theta$ となり、これは $\theta$ に依存します。異なる方向から $(0,0)$ に近づけると異なる値になるため、極限は存在しません。

（3）$\lim_{(x,y)\to(0,0)} \frac{x^{3}+y^{3}}{x^{2}+y^{2}} = 0$

極座標変換により、$\frac{x^{3}+y^{3}}{x^{2}+y^{2}} = r(\cos^{3}\theta + \sin^{3}\theta)$ となります。$|r(\cos^{3}\theta + \sin^{3}\theta)| \le 2r$ かつ $\lim_{r \to 0} 2r = 0$ であるため、はさみうちの原理より極限は $0$ となります。

（4）極限は存在しない

$y = kx$ という経路では極限値は $0$ となりますが、$y^{2} = kx$ という経路では極限値は $\frac{k}{1+k^{2}}$ となります。$k = 0$ のとき $0$、$k = 1$ のとき $\frac{1}{2}$ となり、異なる経路で異なる極限値が得られるため、極限は存在しません。
