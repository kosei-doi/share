【問題】

制約条件 $\phi(x,y)=0$ の下での $f(x,y)$ の最大最小を求めよ。

（2）$\phi(x,y)=x^{3}+y^{3}+x+y-4$、$f(x,y)=xy$

【方針】
必要な知識・テクニック
  • ラグランジュの未定乗数法：制約条件 $\phi(x,y) = 0$ の下で関数 $f(x,y)$ の極値を求める方法です。ラグランジュ関数 $F(x,y,\lambda) = f(x,y) - \lambda \phi(x,y)$ を定義し、$F_{x} = 0$、$F_{y} = 0$、$F_{\lambda} = 0$ を満たす点を求めます。ここで、$\lambda$ はラグランジュ乗数と呼ばれる未定乗数です。
  • 偏微分：2変数関数 $F(x,y,\lambda)$ の $x$ に関する偏微分 $F_{x}$ は、$y$ と $\lambda$ を定数として $x$ で微分したものです。同様に、$F_{y}$ は $x$ と $\lambda$ を定数として $y$ で微分したものです。$F_{\lambda}$ は $x$ と $y$ を定数として $\lambda$ で微分したものです。
  • 連立方程式の解法：$F_{x} = 0$、$F_{y} = 0$、$F_{\lambda} = 0$ の連立方程式を解くことで、極値候補点を求めます。通常、$\lambda$ を消去して $x$ と $y$ の関係式を導き、制約条件と組み合わせて解を求めます。
  • 因数分解公式：$a^{3} - b^{3} = (a-b)(a^{2}+ab+b^{2})$ という因数分解公式を用います。この公式は、$x^{3} - y^{3}$ を $(x-y)(x^{2}+xy+y^{2})$ と因数分解する際に使用します。
  • 平方完成：2次式 $ax^{2}+bx+c$ を $a(x-h)^{2}+k$ の形に変形することを平方完成といいます。これにより、2次式の符号や最小値を調べることができます。
  • 最大値・最小値の判定：候補点での関数値を計算し、さらに制約条件を満たす点での関数の極限を調べることで、最大値・最小値の存在を判定します。極限が負の無限大に発散する場合は最小値が存在しない可能性があります。

【解答】

この問題では、ラグランジュの未定乗数法を用いて、制約条件 $\phi(x,y) = 0$ の下で関数 $f(x,y)$ の最大値・最小値を求めます。まず、ラグランジュ関数を定義し、その偏導関数を計算して連立方程式を解くことで極値候補点を求めます。次に、候補点での関数値を計算し、制約条件を満たす点での関数の極限を調べることで、最大値・最小値の存在を判定します。

（2）$\phi(x,y)=x^{3}+y^{3}+x+y-4$、$f(x,y)=xy$ について

まず、ラグランジュの未定乗数法を用いるため、ラグランジュ関数 $F(x,y,\lambda)$ を定義します。ラグランジュ関数は、目的関数 $f(x,y)$ から制約条件 $\phi(x,y)$ にラグランジュ乗数 $\lambda$ を掛けたものを引いたものです。

$$F(x,y,\lambda) = f(x,y) - \lambda \phi(x,y) = xy - \lambda(x^{3} + y^{3} + x + y - 4)$$

となります。ここで、$\lambda$ はラグランジュ乗数と呼ばれる未定乗数です。

補足： ラグランジュの未定乗数法では、制約条件 $\phi(x,y) = 0$ の下で $f(x,y)$ の極値を求めるために、ラグランジュ関数 $F(x,y,\lambda) = f(x,y) - \lambda \phi(x,y)$ を定義します。この関数の停留点を求めることで、制約条件を満たす $f(x,y)$ の極値候補点が得られます。

次に、ラグランジュ関数 $F(x,y,\lambda)$ の偏導関数を計算します。まず、$x$ に関する偏微分 $F_{x}(x,y,\lambda)$ を計算します。$y$ と $\lambda$ を定数として $x$ で微分します。

$xy$ を $x$ で偏微分すると $y$ です。$-\lambda(x^{3} + y^{3} + x + y - 4)$ について、$-\lambda x^{3}$ を $x$ で偏微分すると $-3\lambda x^{2}$ です。$-\lambda y^{3}$ は $x$ を含まないため、$x$ で偏微分すると $0$ です。$-\lambda x$ を $x$ で偏微分すると $-\lambda$ です。$-\lambda y$ は $x$ を含まないため、$x$ で偏微分すると $0$ です。$-\lambda(-4) = 4\lambda$ は $x$ を含まないため、$x$ で偏微分すると $0$ です。したがって、

$$F_{x}(x,y,\lambda) = y - 3\lambda x^{2} - \lambda$$

となります。

次に、$y$ に関する偏微分 $F_{y}(x,y,\lambda)$ を計算します。$x$ と $\lambda$ を定数として $y$ で微分します。

$xy$ を $y$ で偏微分すると $x$ です。$-\lambda(x^{3} + y^{3} + x + y - 4)$ について、$-\lambda x^{3}$ は $y$ を含まないため、$y$ で偏微分すると $0$ です。$-\lambda y^{3}$ を $y$ で偏微分すると $-3\lambda y^{2}$ です。$-\lambda x$ は $y$ を含まないため、$y$ で偏微分すると $0$ です。$-\lambda y$ を $y$ で偏微分すると $-\lambda$ です。$-\lambda(-4) = 4\lambda$ は $y$ を含まないため、$y$ で偏微分すると $0$ です。したがって、

$$F_{y}(x,y,\lambda) = x - 3\lambda y^{2} - \lambda$$

となります。

次に、$\lambda$ に関する偏微分 $F_{\lambda}(x,y,\lambda)$ を計算します。$x$ と $y$ を定数として $\lambda$ で微分します。

$xy$ は $\lambda$ を含まないため、$\lambda$ で偏微分すると $0$ です。$-\lambda(x^{3} + y^{3} + x + y - 4) = -\lambda x^{3} - \lambda y^{3} - \lambda x - \lambda y + 4\lambda$ について、$-\lambda x^{3}$ を $\lambda$ で偏微分すると $-x^{3}$ です。$-\lambda y^{3}$ を $\lambda$ で偏微分すると $-y^{3}$ です。$-\lambda x$ を $\lambda$ で偏微分すると $-x$ です。$-\lambda y$ を $\lambda$ で偏微分すると $-y$ です。$4\lambda$ を $\lambda$ で偏微分すると $4$ です。したがって、

$$F_{\lambda}(x,y,\lambda) = -x^{3} - y^{3} - x - y + 4$$

となります。

次に、ラグランジュの未定乗数法により、$F_{x} = 0$、$F_{y} = 0$、$F_{\lambda} = 0$ を満たす点を求めます。したがって、連立方程式

$$\begin{cases} F_{x} = y - 3\lambda x^{2} - \lambda = 0 \\ F_{y} = x - 3\lambda y^{2} - \lambda = 0 \\ F_{\lambda} = -x^{3} - y^{3} - x - y + 4 = 0 \end{cases}$$

を解きます。

まず、$F_{x} = 0$ より、

$$y - 3\lambda x^{2} - \lambda = 0$$

となります。$\lambda$ について整理すると、

$$y = 3\lambda x^{2} + \lambda = \lambda(3x^{2} + 1)$$

となります。ここで、$3\lambda x^{2} + \lambda = \lambda(3x^{2} + 1)$ という因数分解を用いました。

同様に、$F_{y} = 0$ より、

$$x - 3\lambda y^{2} - \lambda = 0$$

となります。$\lambda$ について整理すると、

$$x = 3\lambda y^{2} + \lambda = \lambda(3y^{2} + 1)$$

となります。

したがって、連立方程式は、

$$\begin{cases} y = \lambda(3x^{2} + 1) \\ x = \lambda(3y^{2} + 1) \\ x^{3} + y^{3} + x + y - 4 = 0 \end{cases}$$

となります。ここで、$F_{\lambda} = 0$ より $-x^{3} - y^{3} - x - y + 4 = 0$ すなわち $x^{3} + y^{3} + x + y - 4 = 0$ であることを用いました。

次に、$\lambda$ を消去して $x$ と $y$ の関係式を導きます。$y = \lambda(3x^{2} + 1)$ より、$3x^{2} + 1 \ne 0$ のとき、

$$\lambda = \frac{y}{3x^{2} + 1}$$

となります。同様に、$x = \lambda(3y^{2} + 1)$ より、$3y^{2} + 1 \ne 0$ のとき、

$$\lambda = \frac{x}{3y^{2} + 1}$$

となります。

補足： $3x^{2} + 1 \ge 1 > 0$、$3y^{2} + 1 \ge 1 > 0$ であるため、常に $3x^{2} + 1 \ne 0$、$3y^{2} + 1 \ne 0$ です。したがって、$\lambda$ は常に定義できます。

したがって、$\lambda = \frac{y}{3x^{2} + 1}$ と $\lambda = \frac{x}{3y^{2} + 1}$ が同時に成り立つため、

$$\frac{y}{3x^{2} + 1} = \frac{x}{3y^{2} + 1}$$

となります。両辺に $(3x^{2} + 1)(3y^{2} + 1)$ を掛けると、

$$y(3y^{2} + 1) = x(3x^{2} + 1)$$

となります。左辺を展開すると、

$$3y^{3} + y = 3x^{3} + x$$

となります。両辺を移項すると、

$$3y^{3} + y - 3x^{3} - x = 0$$

すなわち、

$$3(x^{3} - y^{3}) + (x - y) = 0$$

となります。ここで、$3y^{3} + y - 3x^{3} - x = 3(y^{3} - x^{3}) + (y - x)$ と変形できます。さらに、$y^{3} - x^{3} = -(x^{3} - y^{3})$、$y - x = -(x - y)$ であるため、$3(y^{3} - x^{3}) + (y - x) = 3(-(x^{3} - y^{3})) + (-(x - y)) = -3(x^{3} - y^{3}) - (x - y)$ となります。したがって、$3y^{3} + y - 3x^{3} - x = -3(x^{3} - y^{3}) - (x - y) = 0$ より、両辺に $-1$ を掛けると、$3(x^{3} - y^{3}) + (x - y) = 0$ となります。

補足： $3y^{3} + y - 3x^{3} - x = 3(y^{3} - x^{3}) + (y - x)$ と因数分解できます。ここで、$y^{3} - x^{3} = -(x^{3} - y^{3})$、$y - x = -(x - y)$ であるため、$3(y^{3} - x^{3}) + (y - x) = -3(x^{3} - y^{3}) - (x - y)$ となります。したがって、$-3(x^{3} - y^{3}) - (x - y) = 0$ の両辺に $-1$ を掛けると、$3(x^{3} - y^{3}) + (x - y) = 0$ となります。

次に、$x^{3} - y^{3}$ を因数分解します。因数分解公式 $a^{3} - b^{3} = (a-b)(a^{2}+ab+b^{2})$ を用いると、

$$x^{3} - y^{3} = (x-y)(x^{2}+xy+y^{2})$$

となります。したがって、

$$3(x-y)(x^{2}+xy+y^{2}) + (x-y) = 0$$

となります。$(x-y)$ で因数分解すると、

$$(x-y)\{3(x^{2}+xy+y^{2}) + 1\} = 0$$

となります。

次に、$3(x^{2}+xy+y^{2}) + 1$ の符号を調べます。$x^{2}+xy+y^{2}$ について、$x$ に関する2次式として平方完成すると、

$$x^{2}+xy+y^{2} = \left(x + \frac{y}{2}\right)^{2} + \frac{3y^{2}}{4}$$

となります。ここで、$\left(x + \frac{y}{2}\right)^{2} \ge 0$、$\frac{3y^{2}}{4} \ge 0$ であるため、$x^{2}+xy+y^{2} \ge 0$ です。等号成立は $x = -\frac{y}{2}$ かつ $y = 0$ すなわち $x = y = 0$ のときのみです。

補足： $x^{2}+xy+y^{2}$ を $x$ について平方完成すると、$x^{2}+xy+y^{2} = x^{2} + xy + \frac{y^{2}}{4} + \frac{3y^{2}}{4} = \left(x + \frac{y}{2}\right)^{2} + \frac{3y^{2}}{4}$ となります。したがって、$x^{2}+xy+y^{2} \ge 0$ であり、等号成立は $x = -\frac{y}{2}$ かつ $y = 0$ すなわち $x = y = 0$ のときのみです。

したがって、$(x,y) \ne (0,0)$ のとき $x^{2}+xy+y^{2} > 0$ です。$(x,y) = (0,0)$ のとき、制約条件 $x^{3}+y^{3}+x+y-4=0$ に代入すると $0 + 0 + 0 + 0 - 4 = -4 \ne 0$ であるため、制約条件を満たしません。したがって、制約条件を満たす点では常に $x^{2}+xy+y^{2} > 0$ です。

したがって、$3(x^{2}+xy+y^{2}) + 1 > 0$ です。したがって、$(x-y)\{3(x^{2}+xy+y^{2}) + 1\} = 0$ より、

$$x - y = 0$$

すなわち、

$$x = y$$

となります。

次に、$x = y$ を制約条件 $x^{3}+y^{3}+x+y-4=0$ に代入します。$y = x$ を代入すると、

$$x^{3} + x^{3} + x + x - 4 = 0$$

となります。整理すると、

$$2x^{3} + 2x - 4 = 0$$

となります。両辺を $2$ で割ると、

$$x^{3} + x - 2 = 0$$

となります。左辺を因数分解します。$x = 1$ を代入すると $1 + 1 - 2 = 0$ であるため、$x - 1$ が因数となります。組立除法または多項式の割り算により、

$$x^{3} + x - 2 = (x-1)(x^{2}+x+2)$$

となります。

補足： $x^{3} + x - 2$ を $x - 1$ で割ると、$x^{2} + x + 2$ が商、$0$ が余りとなります。したがって、$x^{3} + x - 2 = (x-1)(x^{2}+x+2)$ です。

したがって、$x^{3} + x - 2 = 0$ より、

$$(x-1)(x^{2}+x+2) = 0$$

となります。したがって、$x = 1$ または $x^{2}+x+2 = 0$ です。

次に、$x^{2}+x+2 = 0$ の実数解の存在を調べます。$x^{2}+x+2$ を平方完成すると、

$$x^{2}+x+2 = \left(x + \frac{1}{2}\right)^{2} + \frac{7}{4}$$

となります。ここで、$\left(x + \frac{1}{2}\right)^{2} \ge 0$ であるため、$x^{2}+x+2 = \left(x + \frac{1}{2}\right)^{2} + \frac{7}{4} \ge \frac{7}{4} > 0$ です。

補足： $x^{2}+x+2$ を平方完成すると、$x^{2}+x+2 = x^{2} + x + \frac{1}{4} + \frac{7}{4} = \left(x + \frac{1}{2}\right)^{2} + \frac{7}{4}$ となります。したがって、$x^{2}+x+2 \ge \frac{7}{4} > 0$ であり、実数解は存在しません。

したがって、$x^{2}+x+2 = 0$ の実数解は存在しません。したがって、$x = 1$ です。

$x = y = 1$ であるため、極値候補点は、$(x,y) = (1,1)$ です。

次に、候補点での関数値 $f(x,y)$ を計算します。

$(x,y) = (1,1)$ のとき、

$$f(1,1) = 1 \cdot 1 = 1$$

となります。

次に、最大値・最小値の存在を判定するため、制約条件 $x^{3}+y^{3}+x+y-4=0$ を満たす点での関数 $f(x,y) = xy$ の挙動を調べます。

まず、$x \to \infty$ のときを考えます。制約条件 $x^{3}+y^{3}+x+y-4=0$ を満たすためには、$x^{3}$ が大きくなるため、$y^{3}$ が負の大きな値でなくてはなりません。したがって、$y \to -\infty$ でなくてはなりません。

補足： $x \to \infty$ のとき、$x^{3} \to \infty$、$x \to \infty$ であるため、制約条件 $x^{3}+y^{3}+x+y-4=0$ を満たすためには、$y^{3} \to -\infty$ すなわち $y \to -\infty$ でなくてはなりません。

したがって、$x \to \infty$ のとき $y \to -\infty$ であるため、

$$f(x,y) = xy \to \infty \cdot (-\infty) = -\infty$$

となります。ここで、$x \to \infty$、$y \to -\infty$ のとき、$xy \to -\infty$ です。

同様に、$x \to -\infty$ のときを考えます。制約条件 $x^{3}+y^{3}+x+y-4=0$ を満たすためには、$x^{3} \to -\infty$、$x \to -\infty$ であるため、$y^{3} \to \infty$ すなわち $y \to \infty$ でなくてはなりません。

したがって、$x \to -\infty$ のとき $y \to \infty$ であるため、

$$f(x,y) = xy \to (-\infty) \cdot \infty = -\infty$$

となります。ここで、$x \to -\infty$、$y \to \infty$ のとき、$xy \to -\infty$ です。

補足： $x \to \infty$ のときも $x \to -\infty$ のときも、制約条件を満たす点では $f(x,y) = xy \to -\infty$ となります。したがって、制約条件 $x^{3}+y^{3}+x+y-4=0$ の下で $f(x,y)$ は下に有界ではなく、最小値は存在しません。一方、$f(1,1) = 1$ が最大値となります。

したがって、$(x,y) = (1,1)$ のとき最大値 $1$ をとり、最小値は存在しません。

【検証】

計算した内容を検証します。

まず、ラグランジュ関数の計算を確認します。$F(x,y,\lambda) = xy - \lambda(x^{3} + y^{3} + x + y - 4)$ について、$F_{x} = y - 3\lambda x^{2} - \lambda$、$F_{y} = x - 3\lambda y^{2} - \lambda$、$F_{\lambda} = -x^{3} - y^{3} - x - y + 4$ は正しいです。

次に、連立方程式の解法を確認します。$F_{x} = 0$ より $y = \lambda(3x^{2} + 1)$、$F_{y} = 0$ より $x = \lambda(3y^{2} + 1)$ です。これらから $\lambda = \frac{y}{3x^{2}+1} = \frac{x}{3y^{2}+1}$ となり、$y(3y^{2}+1) = x(3x^{2}+1)$ すなわち $3y^{3}+y = 3x^{3}+x$ です。移項すると $3(x^{3}-y^{3})+(x-y)=0$ となり、因数分解すると $(x-y)\{3(x^{2}+xy+y^{2})+1\}=0$ です。$3(x^{2}+xy+y^{2})+1 > 0$ であるため $x = y$ です。制約条件に $y = x$ を代入すると $2x^{3}+2x-4=0$ すなわち $x^{3}+x-2=0$ となり、因数分解すると $(x-1)(x^{2}+x+2)=0$ です。$x^{2}+x+2 = \left(x+\frac{1}{2}\right)^{2}+\frac{7}{4} > 0$ であるため $x = 1$ です。したがって、候補点は $(1,1)$ です。この計算は正しいです。

関数値の計算を確認します。$f(1,1) = 1$ は正しいです。極限の議論について、$x \to \infty$ のとき制約条件を満たすためには $y \to -\infty$ でなくてはならず、$f(x,y) = xy \to -\infty$ です。同様に、$x \to -\infty$ のとき $y \to \infty$ で $f(x,y) \to -\infty$ です。したがって、最小値は存在せず、最大値は $1$ です。この判定は正しいです。

以上より求めた結果の正しさを確認しました。
